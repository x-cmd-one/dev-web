x proxy dnf ​

setting the dnf mirror

Sub Commands ​

NameDescription
x proxy dnf urlProvide url of mirror candidates
x proxy dnf replaceReplace the download mirror
x proxy dnf rollbackRollback the download mirror
x proxy dnf getGet the current mirror source url
x proxy dnf setSet the mirror source, select one from the current mirror source and then set it (TODO: support custom mirror source)
x proxy dnf unsetReset the source to the original official

x proxy dnf url ​

Provide url of mirror candidates

  • Usage :
sh
x proxy dnf url <#1: "ali|ustc|tuna">
x proxy dnf url <#1: "ali|ustc|tuna">

x proxy dnf replace ​

Replace the download mirror

  • Usage :
sh
x proxy dnf replace|set <#1: "ali|ustc|tuna">
x proxy dnf replace|set <#1: "ali|ustc|tuna">

x proxy dnf rollback ​

Rollback the download mirror

  • Usage :
sh
x proxy dnf rollback [SUB_COMMAND] <#1: "use the selected file to rollback">
x proxy dnf rollback [SUB_COMMAND] <#1: "use the selected file to rollback">
  • Sub Commands :
NameDescription
x proxy dnf rollback lslist all file you can rollback

x proxy dnf rollback ls ​

list all file you can rollback

  • Usage :
sh
x proxy dnf rollback ls
x proxy dnf rollback ls
